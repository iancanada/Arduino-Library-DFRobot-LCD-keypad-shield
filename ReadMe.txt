DRF0009.h and DRF0009.cpp is DFRobot Ardunino C++ library:

/DRF0009Demo folder is the demo code of how to use this library

Demo code and library tested on Arduino UNO and MEGA2560

If name of each pressed button can be shown on the LCD screen, then we are done

DFRobot DFR0009 LCD keypad shield can be sourced at:
http://www.robotshop.com/en/dfrobot-lcd-keypad-shield-arduino.html
